# Pharmaceutical Management System

## Overview

The **Pharmaceutical Management System** is a web application designed to help manage prescriptions, inventory, and user accounts for a pharmacy. The system allows users to log in, register, and interact with various features related to managing prescriptions and pharmacists.

## Features

- **User Authentication**: Users can log in or register to access the system.
- **Prescription Management**: Users can view and manage prescriptions.
- **Pharmacist Management**: Assign pharmacists to prescriptions and track fulfillment.
- **Responsive Design**: The UI is designed to work seamlessly on both desktop and mobile devices.

## Technologies Used

- **React**: Frontend UI library used to build interactive components and manage application state.
- **React Router**: Used for routing and navigation between pages.
- **Tailwind CSS**: Utility-first CSS framework used for styling the app.
- **JavaScript (ES6+)**: For logic and functionality.
  
## Setup and Installation

### Prerequisites

Ensure you have the following installed:
- **Node.js** (version 14 or higher)
- **npm** (Node package manager)

### Steps to Install

1. **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/pharmaceutical-management-system.git
    ```

2. **Navigate to the project directory:**
    ```bash
    cd pharmaceutical-management-system
    ```

3. **Install dependencies:**
    ```bash
    npm install
    ```

4. **Start the development server:**
    ```bash
    npm start
    ```
    This will start the app on `http://localhost:3000`.

## Folder Structure

- **src/**: Contains all the main source code.
  - **assets/**: Folder containing all the images, such as `home.jpg`.
  - **components/**: Reusable components for the app.
  - **pages/**: React components for different pages (Home, Login, Register, etc.).
  - **App.jsx**: The main entry point of the application where routing is defined.
  - **index.jsx**: The root file where the React app is rendered.
